# Shared

Shared resources usable across features per ERP.md §1.2.

- `components/` — reusable UI building blocks (atoms, molecules, organisms)
- `hooks/` — generic React hooks
- `stores/` — global Zustand stores not tied to one feature
- `services/` — cross-feature services
- `utils/` — pure helpers
- `types/` — shared TypeScript types
- `constants/` — app-wide constants
